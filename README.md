# To-Do List

Sade ve kullanışlı bir yapılacaklar listesi uygulaması. Vanilla JavaScript ile geliştirilmiştir.

## Özellikler

- Görev ekleme (buton veya Enter tuşu ile)
- Checkbox ile görevi tamamlandı olarak işaretleme
- Tek tek görev silme
- Reset butonu ile tüm listeyi temizleme
- LocalStorage ile sayfa yenilense bile görevler kaybolmaz

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript (Vanilla)
- Vite (geliştirme aracı)

## Kurulum

```bash
npm install
npm run dev
```

Tarayıcıda `http://localhost:5173` adresini aç.
